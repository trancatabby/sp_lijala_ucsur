Lijala is a radical keyboard for the sitelen pona writing system (specifically the UCSUR standard) that favors efficiency over accuracy. It was created out of frustration with a different keyboard layout, nasin Wakalito, for having lengthy key combinations that were personally hard for me to follow. (Lijala is not meant to compete with Wakalito, and if you prefer it, that's fine!)

Because of how Lijala is designed, every word has only one way to type it, and they can get rather arbitrary depending on what was available during development. Here, the glyphs will be split up into sections, each having their own "shape." Each section has 3 columns: the word's glyph, how the word is typed in the original AutoHotKey Lijala, and how the word is typed on Keyman Lijala if it's been changed. (The original included nimisin only compatible with nasin nanpa, so some words have been adjusted to preserve fluidity.)
